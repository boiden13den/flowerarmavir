# Зефирные Букеты Армавир

Современный интернет-магазин дизайнерских букетов из зефира на Vue 3 с расширенным функционалом.

## 🚀 Технологии

- **Vue 3** (Composition API + `<script setup>`)
- **Vue Router 4** - маршрутизация
- **Pinia** - управление состоянием
- **Vite** - сборка проекта
- **TailwindCSS** - стилизация
- **Axios** - HTTP-клиент

## 📦 Установка

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр production сборки
npm run preview
```

## 🏗️ Структура проекта

```
src/
├── assets/          # Стили и ресурсы
├── components/      # Переиспользуемые компоненты
├── composables/     # Composition functions
├── layouts/         # Базовые лейауты
├── pages/           # Компоненты страниц
├── router/          # Настройка роутера
├── stores/          # Pinia stores
├── utils/           # Вспомогательные функции
└── App.vue + main.js
```

## 📄 Страницы

## 🔧 Настройка

### Переключение на реальный API

В файле `src/utils/api.js` измените:

```javascript
const USE_MOCK = false
```

И настройте baseURL для вашего бэкенда.

### Данные

Mock-данные находятся в `public/data/`:
- `products.json` - товары
- `faq.json` - вопросы и ответы
- `reviews.json` - отзывы
- `blog.json` - статьи блога
- `promotions.json` - акции

## 🔐 Авторизация

Система авторизации работает через localStorage (mock):
- Регистрация новых пользователей
- Вход в систему
- Защищенные маршруты (profile, orders)
- Выход из системы

Данные пользователей хранятся в `localStorage.registeredUsers`
